- Sample： 样本名称；
- Total reads： 总 reads 数量；
- Mapped reads： 比对至参考基因组上的 reads 数量（包括单端比对和双端比对）；
- Mapping rate： 比对率，比对至参考基因组上的 reads 数量占总 reads 数量的百分比；
- Dup. nums： 重复序列 reads 数量；
- Dup. rate： 重复序列 reads 数量占总 reads 数量的百分比。
